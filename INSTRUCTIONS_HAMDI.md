# Instructions pour Hamdi - Accès Panneau Admin

Salut Hamdi ! Voici comment accéder au panneau d'administration du site.

## Étape 1 : Créer ton compte

1. Va sur le site web (l'URL que ton ami t'a partagé)
2. Clique sur **"Se connecter"** en haut à droite
3. Clique sur **"Créer un compte"**
4. **IMPORTANT** : Utilise l'email **hamditebourbi07@gmail.com**
5. Choisis un mot de passe sécurisé
6. Clique sur **"S'inscrire"**

## Étape 2 : Accéder au panneau admin

Une fois connecté avec ton compte :

1. Dans la barre d'adresse de ton navigateur, ajoute `/admin/dashboard` à la fin de l'URL
   - Exemple : `https://ton-site.com/admin/dashboard`
2. Appuie sur Entrée
3. **Tu es maintenant administrateur !** 🎉

## Que peux-tu faire en tant qu'admin ?

### 📦 Gérer les produits
- Ajouter de nouveaux produits
- Modifier les produits existants
- Supprimer des produits
- Télécharger des images de produits

### 🏷️ Gérer les catégories

### 📊 Tableau de bord
- Voir les statistiques de ventes
- Consulter les commandes récentes
- Suivre le chiffre d'affaires

### ⚙️ Paramètres du site
- Modifier le nom de la boutique
- Changer le logo
- Personnaliser l'image de la page d'accueil
- Modifier les informations de contact
